= Mettez à jour les informations d'identification du serveur vCenter.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Vous pouvez mettre à jour les informations d'identification de l'instance vCenter Server à l'aide de la console de maintenance.

.Avant de commencer
Vous devez disposer des informations de connexion de l'utilisateur de maintenance.

*À propos de cette tâche*

Si vous avez modifié les informations d'identification de vCenter Server après le déploiement, vous devez les mettre à jour en suivant cette procédure.

.Étapes
. Depuis vCenter Server, ouvrez une console sur VASA Provider.
. Connectez-vous en tant qu'utilisateur de maintenance.
. Entrer `2` pour sélectionner le menu de configuration du système.
. Entrer `8` modifier les informations d'identification vCenter.
